📱 Dark Mode & Mobile Integration

✅ How to Add Dark Mode & Mobile Responsiveness

Step 1: Add CSS Link

Add this line in the <head> section of your HTML:

<link rel="stylesheet" href="css/dark-mode.css">

Step 2: Add JavaScript

Add this before closing </body> tag:

<script src="js/dark-mode.js"></script>

Step 3: Add Toggle Button (Optional)

Add this button anywhere you want the dark mode toggle:

<button class="theme-toggle">🌙 Dark Mode</button>
<!-- OR -->
<button data-theme-toggle>🌙 Dark Mode</button>

Step 4: Use CSS Variables in Your Styles

Replace your color values with CSS variables:

/* OLD */
.box {
    background-color: #ffffff;
    color: #333333;
    border: 1px solid #dddddd;
}

/* NEW - Automatically works in both light & dark mode */
.box {
    background-color: var(--bg-primary);
    color: var(--text-primary);
    border: 1px solid var(--border-color);
}
