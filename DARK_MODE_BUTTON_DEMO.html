🌓 Dark Mode - Clickable Button Demo

Click the button at the top right or press Ctrl+Shift+D to toggle dark mode

✨ Button Features

  • ✓ Click to toggle dark/light mode
  • ✓ Icon and text update automatically
  • ✓ Smooth animations on hover
  • ✓ Remembers your preference
  • ✓ Works on mobile and desktop
  • ✓ Keyboard shortcut: Ctrl+Shift+D

📋 How to Add to Your Pages

Step 1: Add CSS (in <head>)

<link rel="stylesheet" href="css/dark-mode.css">

Step 2: Add Button (anywhere in <body>)

<button class="dark-mode-toggle" onclick="toggleDarkMode()">
    <span class="toggle-icon">🌙</span>
    <span class="toggle-text">Dark</span>
</button>

Step 3: Add JavaScript (before </body>)

<script src="js/dark-mode.js"></script>

🎨 Button Styles

Option 1: Fixed Position (Top Right)

.dark-mode-toggle {
    position: fixed;
    top: 20px;
    right: 20px;
    z-index: 9999;
}

Option 2: In Navbar

<!-- Add button to your navbar -->
<nav>
    <button class="dark-mode-toggle" onclick="toggleDarkMode()">
        <span class="toggle-icon">🌙</span>
        <span class="toggle-text">Dark</span>
    </button>
</nav>

Option 3: Simple Style

<button class="theme-toggle" onclick="toggleDarkMode()">
    🌙 Dark Mode
</button>
