Problem: Given a 1-indexed sorted array, find two numbers that add up to a target value and return their 1-based indices.

💡 Key Insight: Array is sorted! Use two pointers from opposite ends - increase left if sum is too small, decrease right if too large.
Approach: Two Pointers
  • Start with left pointer at beginning, right pointer at end
  • Calculate sum = numbers[l] + numbers[r]
  • If sum equals target, return indices (1-based)
  • If sum is too small, move left pointer right to increase sum
  • If sum is too large, move right pointer left to decrease sum
⏱️ Time
O(n)
💾 Space
O(1)
Solution
def two_sum_sorted(numbers, target):
    l, r = 0, len(numbers) - 1
    while l < r:
        s = numbers[l] + numbers[r]
        if s == target:
            return [l+1, r+1]  # 1-based indexing
        elif s < target:
            l += 1
        else:
            r -= 1
    return []
📋 Example: [2,7,11,15], target=9
Array: [2, 7, 11, 15] Target: 9 Initial: l=0(2), r=3(15) Sum: 2 + 15 = 17 > 9 → Move right Step 2: l=0(2), r=2(11) Sum: 2 + 11 = 13 > 9 → Move right Step 3: l=0(2), r=1(7) Sum: 2 + 7 = 9 ✓ FOUND! Answer: [1, 2] (1-based indices)

Problem: Two numbers are represented as linked lists in REVERSE order. Add them and return the sum as a linked list.

💡 Key Insight: Traverse both lists simultaneously, handle carry at each step, continue until both lists are exhausted.
Approach: Single Pass with Carry
  • Create a dummy node to simplify code
  • Initialize carry = 0
  • Iterate while either list has nodes or carry exists
  • Get values from both lists (0 if list is None)
  • Add values with carry, create new node with digit, update carry
  • Return dummy.next
⏱️ Time
O(max(m,n))
💾 Space
O(max(m,n))
Solution
def add_two_numbers(l1, l2):
    dummy = ListNode(0)
    curr = dummy
    carry = 0
    
    while l1 or l2 or carry:
        v1 = l1.val if l1 else 0
        v2 = l2.val if l2 else 0
        total = v1 + v2 + carry
        carry = total // 10
        digit = total % 10
        
        curr.next = ListNode(digit)
        curr = curr.next
        l1 = l1.next if l1 else None
        l2 = l2.next if l2 else None
    
    return dummy.next
📋 Example: 342 + 465 = 807
List 1: 2 -> 4 -> 3 (represents 342) List 2: 5 -> 6 -> 4 (represents 465) Step 1: 2 + 5 = 7, no carry Result: 7 Step 2: 4 + 6 = 10, carry = 1, digit = 0 Result: 7 -> 0 Step 3: 3 + 4 + 1(carry) = 8, no carry Result: 7 -> 0 -> 8 Answer: 7 -> 0 -> 8 (represents 807)

Problem: Find the length of the longest substring that contains no repeating characters.

💡 Key Insight: Use sliding window with a set to track characters in current window. When duplicate found, shrink from left.
Approach: Sliding Window
  • Maintain left and right pointers for window
  • Use a set to store characters in current window
  • Expand window: move right pointer and add character
  • If duplicate found, shrink from left until no duplicate
  • Track maximum window length throughout
⏱️ Time
O(n)
💾 Space
O(min(m,n))
Solution
def length_of_longest_substring(s):
    seen = set()
    left = 0
    max_len = 0
    
    for right, ch in enumerate(s):
        while ch in seen:
            seen.remove(s[left])
            left += 1
        
        seen.add(ch)
        max_len = max(max_len, right - left + 1)
    
    return max_len
📋 Example: "abcabcbb"
String: a b c a b c b b Window: "a" → length = 1 Window: "ab" → length = 2 Window: "abc" → length = 3 (MAX) Duplicate 'a': remove from left Window: "bca" → length = 3 Duplicate 'b': remove from left Window: "cab" → length = 3 Duplicate 'c': remove from left Window: "ab" → length = 2 Duplicate 'b': remove from left Window: "b" → length = 1 Answer: 3 (substring "abc")

Problem: Find all unique triplets in array that sum to zero.

💡 Key Insight: Sort first! Then fix one element and use two pointers for remaining two. Skip duplicates carefully.
Approach: Sort + Two Pointers
  • Sort the array first
  • For each element i, find two numbers that sum to -numbers[i]
  • Use two pointers (left, right) for remaining elements
  • Skip duplicate values at each level to avoid duplicate triplets
  • Move pointers based on sum comparison
⏱️ Time
O(n²)
💾 Space
O(1)
Solution
def three_sum(nums):
    nums.sort()
    res = []
    n = len(nums)
    
    for i in range(n - 2):
        if i > 0 and nums[i] == nums[i-1]:
            continue
        if nums[i] > 0:
            break
        
        l, r = i + 1, n - 1
        target = -nums[i]
        
        while l < r:
            s = nums[l] + nums[r]
            if s == target:
                res.append([nums[i], nums[l], nums[r]])
                while l < r and nums[l] == nums[l+1]:
                    l += 1
                while l < r and nums[r] == nums[r-1]:
                    r -= 1
                l += 1
                r -= 1
            elif s < target:
                l += 1
            else:
                r -= 1
    
    return res

Original: [-1, 0, 1, 2, -1, -4] Sorted: [-4, -1, -1, 0, 1, 2] Fix i=0 (-4): Need two nums summing to 4 l=1(-1), r=5(2): sum=-3 < 4 (move left) Fix i=1 (-1): Need two nums summing to 1 l=2(-1), r=5(2): sum=1 ✓ FOUND [-1,-1,2] Skip duplicates, continue... Fix i=2 (-1): Already processed, skip duplicate Fix i=3 (0): Need two nums summing to 0 l=4(1), r=5(2): sum=3> 0 (move right) Answer: [[-1,-1,2], [-1,0,1]]

Problem: Group words that are anagrams of each other.

💡 Key Insight: Anagrams have the same letters! Sort each word - identical sorted forms = anagrams. Use sorted form as hash key.
Approach: Hash Map with Sorted Key
  • Create a dictionary mapping sorted word → list of anagrams
  • For each word, sort its characters to get canonical form
  • Use sorted form as key and append word to value list
  • Return all groups (dictionary values)
⏱️ Time
O(n*k log k)
💾 Space
O(n)
Solution
from collections import defaultdict

def group_anagrams(strs):
    d = defaultdict(list)
    
    for s in strs:
        key = tuple(sorted(s))
        d[key].append(s)
    
    return list(d.values())
📋 Example: ["eat", "tea", "ate", "tab", "bat"]
Word "eat": sorted = "aet" Word "tea": sorted = "aet" ✓ Same as "eat" Word "ate": sorted = "aet" ✓ Same as "eat" Word "tab": sorted = "abt" Word "bat": sorted = "abt" ✓ Same as "tab" Grouping: Key "aet": ["eat", "tea", "ate"] Key "abt": ["tab", "bat"] Answer: [["eat", "tea", "ate"], ["tab", "bat"]]
